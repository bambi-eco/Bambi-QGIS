about=A comprehensive QGIS plugin for wildlife detection and tracking in drone videos.
    Features include:
    - Frame extraction from drone videos with configurable sample rate
    - Animal detection using YOLO models
    - Geo-referencing of detections using DEM and camera metadata
    - Multi-object tracking with configurable parameters
    - Automatic creation of polygon layers for tracks in QGIS
    - Automatic segmentation of objects using SAM3
